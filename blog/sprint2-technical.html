Positioning Properties in CSS

Relative

Say if we have to move an element on the Web page, we can do so using something known as a position attribute.
In our example, we intend to move the element down by 50 pixels. Example: Attempting to move the element down without using the position property.


As shown in the example above the element didn't move. To fix this, we will add a position property known as relative.This will move the element down by 50px as we intend to.
The key word relative moves the element relative to the position attribute, in this case 50px down - super confusing but yet makes sense

Absolute

If we want to position an element within another element (A div inside another div), we can use a property called absolute. This is an extemely powerful type of positioning that allows us to literally place any page element exactly where we want it using the positioning attributes top, left, bottom, and right. A massive Gotcha is that the positioning will be relative to the next parent. If there is no such parent, it will default all the way back up to the element itself meaning it will be placed relative to the page itself.
Confused? Let's try to illustrate this concept using examples. The image below shows two divs, one nested (a child) within the other. In this case, we can use absolute which positions the element exactly where we want it to be
